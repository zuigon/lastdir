Lastdir
=======
Simple CLI dir bookmark script

Install
-------
  curl http://github.com/bkrsta/lastdir/raw/master/lastdir.rb > ~/bin/b

Usage
-----

  b [options]

  Options:
    list ls l
    remove rm delete del r
    add a
    separator sep s
